    What is the farm?

    The farm is a background worker on your NAS that pre-generates the previews, proxies, and transcripts your media needs — so OpenLoupe and this web UI can show thumbnails, scrub clips, and search speech instantly instead of regenerating everything on your laptop.

    Generate on the farm

    Queue a folder or file for the NAS to pre-generate previews, proxies, and transcripts in the background. Jobs run as soon as a farm worker is online.

          Admin key rotation
          
              Re-encrypts every saved Destination under a new admin key. After success, you MUST update JM_ADMIN_KEY in the container env and restart — the in-memory key stays on the OLD value until restart so the on-disk state stays decryptable through the rotation window. Wrong old key returns 401 without touching state (verify-before-rewrite contract).
